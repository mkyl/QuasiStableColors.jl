Quasi-Stable Coloring

Graph compression for performant approximations

QuasiStableColors.jl is a library for compressing graphs and approximating graph algorithms. The compressed graphs are computed using an algorithm called quasi-stable coloring, which results in a much smaller graph while preserving its key properties. This approach is introduced in the research paper "Quasi-stable Coloring for Graph Compression."

A major advantage of this approach is that many algorithms can be computed directly on the compressed graph, without needing decompression. This results in an effective approximation of many graph algorithms. Applications implemented in this library are:

  • Betweenness centrality
  • Maximum-flow/minimum-cut
  • Linear optimization

Citation Format

If you use this algorithm, we ask that you cite our paper:

@misc {
    title = "Quasi-stable Coloring for Graph Compression: Approximating Max-Flow, Linear Programs, and Centrality"
    author = "Moe Kayali and Dan Suciu"
}

Q-stable coloring was developed in the Allen School of Computer Science at the University of Washington, Seattle.
